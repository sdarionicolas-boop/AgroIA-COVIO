# Contribuyendo a AgroIA-COVIO

¡Gracias por tu interés en colaborar con AgroIA-COVIO! Para mantener el proyecto de forma sostenible y garantizar su viabilidad comercial a futuro, todos los contribuidores deben seguir estas pautas.

## Proceso de Contribución

1. **Haz un Fork** del repositorio.
2. **Crea una rama de desarrollo** para tus modificaciones (`git checkout -b feature/nueva-mejora` o `git checkout -b fix/correccion-error`).
3. **Realiza los cambios** asegurándote de no romper la compatibilidad y de probar la aplicación localmente.
4. **Envía un Pull Request (PR)** dirigido a la rama `main` del repositorio original.

## Acuerdo de Licencia de Contribuidor (CLA)

Para poder integrar tu aporte al código base de AgroIA-COVIO, es obligatorio que aceptes los términos del [Acuerdo de Licencia de Contribuidor (CLA)](file:///e:/Desktop_Movidos/PlantDetector/CLA.md).

Este acuerdo nos permite mantener AgroIA-COVIO libre y abierto bajo la licencia GNU GPLv3 para los pequeños y medianos productores, al mismo tiempo que habilita la vía de financiamiento comercial del proyecto mediante el esquema de licenciamiento dual.

**Al realizar un Pull Request en este repositorio, declaras haber leído y aceptado el acuerdo estipulado en [CLA.md](file:///e:/Desktop_Movidos/PlantDetector/CLA.md).**
